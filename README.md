## 🚀 Быстрый старт

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/MADAO81/Bender_Telegram_Bot_Russian.git
2. Установите зависимости
bash

3. Получите токены
Что нужно	Где взять

OPENAI_API_KEY	В личном кабинете OpenAI Platform
GROUP_CHAT_ID	Добавьте бота в группу и отправьте любое сообщение, бот напишет ID в консоль (или используйте @get_my_id_bot)
4. Настройте переменные
Отредактируйте файл bender_bot.py:

⚙️ Конфигурация
Все настройки находятся в верхней части файла bender_bot.py.

Параметр	Значение по умолчанию	Описание
CHANCE_TO_INTERRUPT	0.15	Вероятность влезть в чужой разговор (15%)
GROUP_CHAT_ID	-100...	ID группы, куда Бендер будет писать шутки
Рабочее время	9:00 – 18:00, пн–пт	Можно изменить в функции is_working_hours()
Время шутки	12:00	Ежедневная отправка случайной фразы
🎛️ Как изменить «настроение» Бендера
python
CHANCE_TO_INTERRUPT = 0.15  # 15% — рекомендовано
Значение	Поведение
0.05	Редко (каждое 20-е сообщение)
0.15	Умеренно (оптимально)
0.30	Часто (почти каждое 3-е)

❓ Частые вопросы (FAQ)
Сколько стоят 5$ на OpenAI API?

~25 000–30 000 сообщений в модели GPT-4o Mini. Этого хватит на месяцы активного использования.

Почему карта «Мир» не подходит?

OpenAI принимает только Visa/Mastercard международных банков. Используйте виртуальную карту через сервисы-посредники.

Бот отвечает сам себе?

Нет, в коде есть проверка message.from_user.id == bot.get_me().id, исключающая зацикливание.

Как отключить расписание?

Удалите или закомментируйте вызовы schedule.every()... и threading.Thread.

Как сделать бота тише?

Уменьшите CHANCE_TO_INTERRUPT до 0.05 или 0.03.
