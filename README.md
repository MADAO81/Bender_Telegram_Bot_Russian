# 🤖 Bender Telegram Bot

[![Python 3.9+](https://img.shields.io/badge/python-3.9+-blue.svg)](https://www.python.org/downloads/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![OpenAI](https://img.shields.io/badge/OpenAI-API-412991.svg)](https://openai.com/)

**Bender Bending Rodriguez** из «Футурамы» теперь живёт в вашем Telegram-чате. Он хамит, комментирует картинки, влезает в разговоры без спроса, меняет настроение в зависимости от времени суток и реагирует на триггерные слова. Шутит спонтанно, но в рамках лимитов.

> *"Bite my shiny metal ass!"* — Bender

---

## 📋 Возможности

| Функция | Описание |
| :--- | :--- |
| 💬 **Комментирует сообщения** | С вероятностью 10% влезает в любой диалог (даже если не звали) |
| 🖼️ **Смотрит картинки** | Комментирует любые фото, отправленные в чат (через GPT-4 Vision) |
| 🎯 **Реагирует на упоминания** | Если написать `@BenderBot` или ответить на его сообщение — ответит всегда |

| 🎭 **Система настроения** | Меняет характер в зависимости от времени и дня недели |

| 📊 **Лимит шуток** | Не более 20 шуток в неделю и 15 минут между ними |

☁️ Деплой на VPS
Рекомендуется запускать бота на VPS (например, Beget, SprintBox, Timeweb). Инструкция по настройке:

1. Подключитесь к серверу по SSH
bash
ssh root@IP_АДРЕС_СЕРВЕРА
2. Установите Python и Git
bash
apt update && apt upgrade -y
apt install python3 python3-pip python3-venv git -y
3. Склонируйте репозиторий
bash
git clone https://github.com/ваш-аккаунт/bender-telegram-bot.git
cd bender-telegram-bot
4. Настройте виртуальное окружение и установите зависимости
bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
5. Создайте файл .env с токенами
bash
nano .env
Вставьте:

env
TELEGRAM_TOKEN=ваш_токен_бота
OPENAI_API_KEY=sk-proj-ваш_ключ_openai
Сохраните: Ctrl+O, затем Ctrl+X.

6. Запустите бота
bash
python bender_bot.py
7. Настройте автозапуск (systemd)
Создайте файл сервиса:

bash
nano /etc/systemd/system/benderbot.service
Вставьте:

ini
[Unit]
Description=Bender Telegram Bot
After=network.target

[Service]
User=root
WorkingDirectory=/root/bender-telegram-bot
ExecStart=/root/bender-telegram-bot/venv/bin/python /root/bender-telegram-bot/bender_bot.py
Restart=always

[Install]
WantedBy=multi-user.target
Активируйте и запустите:

bash
systemctl daemon-reload
systemctl enable benderbot
systemctl start benderbot
systemctl status benderbot

❓ Частые вопросы (FAQ)
Сколько стоят 5$ на OpenAI API?

~25 000–30 000 сообщений в модели GPT-4o Mini. Этого хватит на месяцы активного использования.

Почему карта «Мир» не подходит?

OpenAI принимает только Visa/Mastercard международных банков. Используйте виртуальную карту через сервисы-посредники.

Бот отвечает сам себе?

Нет, в коде есть проверка update.message.from_user.id == context.bot.id, исключающая зацикливание.

Почему бот не шутит по расписанию?

Мы убрали расписание. Бендер сам решает, когда пошутить (10% шанс на каждое сообщение), но в рамках лимита 20 шуток в неделю и 15 минут между ними.

Как сделать бота активнее?

Увеличьте CHANCE_TO_JOKE до 0.15 или 0.20 в bender_bot.py.

Как добавить новые шутки?

Отредактируйте файл jokes/jokes_bank.py или расширьте триггерные реакции в jokes/triggers.py.
