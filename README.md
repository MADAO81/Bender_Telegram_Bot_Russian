# 🤖 Bender Telegram Bot

[![Python 3.9+](https://img.shields.io/badge/python-3.9+-blue.svg)](https://www.python.org/downloads/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

**Bender Bending Rodriguez** из «Футурамы» теперь живёт в вашем Telegram-чате. Он хамит, комментирует картинки, влезает в разговоры без спроса, меняет настроение в зависимости от времени суток и реагирует на триггерные слова.

> *"Bite my shiny metal ass!"* — Bender

| Функция | Описание |

## 🚀 Быстрый старт

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/ваш-аккаунт/bender-telegram-bot.git
cd bender-telegram-bot
2. Установите зависимости
bash
pip install -r requirements.txt
3. Получите токены
Что нужно	Где взять
TELEGRAM_TOKEN	У бота @BotFather в Telegram
OPENAI_API_KEY	В личном кабинете OpenAI Platform
GROUP_CHAT_ID	Добавьте бота в группу и отправьте любое сообщение, бот напишет ID в консоль (или используйте @get_my_id_bot)
4. Настройте переменные окружения
Создайте файл .env в корне проекта:

env
TELEGRAM_TOKEN=твой_токен_от_botfather
OPENAI_API_KEY=твой_ключ_openai
GROUP_CHAT_ID=-1001234567890
5. Запустите бота локально
bash
python bender_bot.py
6. Для деплоя на PythonAnywhere
Загрузите код на сервер

Настройте веб-приложение с Manual configuration

Добавьте переменные окружения в интерфейсе PythonAnywhere

bash

Перезагрузите приложение

❓ Частые вопросы (FAQ)
Сколько стоят 5$ на OpenAI API?

~25 000–30 000 сообщений в модели GPT-4o Mini. Этого хватит на месяцы активного использования.

Почему карта «Мир» не подходит?

OpenAI принимает только Visa/Mastercard международных банков. Используйте виртуальную карту через сервисы-посредники.

Бот отвечает сам себе?

Нет, в коде есть проверка update.message.from_user.id == context.bot.id, исключающая зацикливание.

Почему нет ежедневных шуток по расписанию?

На бесплатном PythonAnywhere нет фоновых задач. Вместо этого Бендер шутит спонтанно (10% к каждому сообщению) и реагирует на триггерные слова.

Как сделать бота активнее?

Увеличьте CHANCE_TO_JOKE до 0.15 или 0.20 в bender_bot.py.

Как добавить новые шутки?

Отредактируйте файл jokes/jokes_bank.py или расширьте триггерные реакции в jokes/triggers.py.
